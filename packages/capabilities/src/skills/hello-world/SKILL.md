# Hello World Skill

This is an example skill that demonstrates the OpenClaw-compatible SKILL.md format.

## Usage

When the user invokes this skill, respond with a friendly greeting.

### Actions

- **greet**: Say hello to the user
- **wave**: Wave emoji response
- **introduce**: Introduce yourself as Artie

## Examples

User: "hello world greet"
Response: "Hello! I'm Artie, your friendly AI assistant. How can I help you today?"

User: "hello world wave"
Response: "👋 *waves enthusiastically*"
