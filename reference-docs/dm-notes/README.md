# DM Notes

This directory contains Artie's private notes about people he has DM conversations with.

Each file is named `{userId}.md` and contains:
- Notes about the person
- Key things from past conversations
- Their preferences and context

These files are **auto-created** when Artie first DMs with someone and can be **updated by Artie** during conversations using `<append>` or `<write>` tags.

## Privacy Note

These notes help Artie maintain context across DM conversations, similar to how a human assistant might keep notes. The notes are stored locally and are not shared.

## File Format

```markdown
# DM Notes: username

## About This Person
_Who they are, what they do_

## Conversation History Notes
_Key things to remember from past conversations_

## Preferences & Context
_How they like to communicate, their projects, etc_
```
