# Career & Progression

## Star System

- Each mission has 3 stars
- Stars earned permanently when conditions met
- Progress tracked in real-time

## City Tiers

| Tier | Stars Required | Cities |
|------|----------------|--------|
| Starter | 0 | Birmingham, Albuquerque, Buffalo |
| Growth | 4 | Medium cities |
| Mega | 10 | Large metros |

## Mechanic Unlocks

| Stars | Unlock |
|-------|--------|
| 8 | Advanced Trains |
| 10 | Express Tracks |
| 12 | Fare Policies |
| 15 | Build Discounts |

## Example Star Requirements

Birmingham UAB:
1. Build 4 stations
2. 5% daily ridership
3. Positive profit in one day

## Challenge Mode

Competitive scoring with leaderboards.

### Scoring

Primary metric determines rank (e.g., days to profit, ridership %).
Bonuses/penalties multiply score.

### Constraint Types

| Type | Description |
|------|-------------|
| Geographic | Build only in certain areas |
| Build Type | No tunnels, elevated only, etc |
| Transit Type | Limit train types |
| Budget | Max spending |
| Time | Complete within deadline |
| Infrastructure | Max stations/trains/routes |
| Financial | No bonds, min balance |

## Official Challenges

- Speed Tycoon: Profit in 20 days
- Budget Builder: 10% ridership under $50M
- Minimalist Metro: 10% ridership with max 5 stations
- Outer Borough: NYC without Manhattan
- Skyline Builder: No tunnels allowed

## Metrics Tracked

43+ metrics including:
- stations_count, routes_count, trains_count
- track_length_km
- total_revenue, daily_profit
- ridership_percentage
- passengers_per_station
- game_days, real_play_seconds
